# EvoSlot — site de prezentare

Landing/presentation site for **EvoSlot**, a booking & operations platform for
service businesses (barbershops, salons, spas, beauty clinics). A product of
**EvoNode CO**.

Static site (HTML + CSS + JS), no build step. Deployed on Vercel.

## Local preview
```bash
npx serve .
```

## Structure
- `index.html` — the page
- `styles.css` — design system (Modern SaaS / gradient bold)
- `script.js` — nav, mobile menu, scroll reveals
- `assets/` — product screenshots (prod website + client mobile app)
